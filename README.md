# ロックバンド演奏実践知の構造化・学習支援LOD (PoC)

本プロジェクトは、ロックバンドの演奏において極めて重要でありながら、これまで体系化されてこなかった**「音作り・奏法・機材」**という**実践知**を、LOD（Linked Open Data）として構造化し、効率的な学習支援を目指す試みです。

## 1. 研究の背景と社会的意義
現在、Web上には優れた奏法解説サイトや動画が数多く存在しますが、それらの情報は**断片的かつ散在**しています。
* **課題**: 楽曲のどのタイミングでどの奏法を使い、どのような機材設定でその音を再現するのかという情報は、既存のバンドスコアやWeb検索だけでは統合的に把握できません。
* **機会損失の解消**: この情報の乖離は、初心者にとっての大きな障壁となり、楽器演奏を断念する原因となっています。
* **解決策**: 本研究では、これら散在する情報をRDF形式で構造化し、一箇所に集約・関連付けることで、体系的な学習環境を提示します。

## 2. 公開データの内容 (Demo Data)
本リポジトリでは、本モデルの有用性を検証するための概念実証（PoC）として、以下のデモデータを公開しています。

* **対象楽曲**: SHISHAMO「明日も」
* **重点パート**: ギターパート
* **データの特徴**:
    * **YouTube音源の活用**: 権利関係および実用性の観点から、既存の動画配信サイト（YouTube）の音源をリソースとして配置しています。
    * **機材設定（例）**: アンプやつまみの詳細設定については、公式データが存在しないため、本研究における推奨設定を例として記入しています。
    * **拡張性**: 本データはギター中心ですが、モデル自体はベース、ドラムといった全パート、および他楽曲へ容易に拡張可能な設計となっています。
