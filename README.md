# ロックバンド演奏実践知の構造化LOD

本リポジトリは、ロックバンドの演奏において重要な「音作り・奏法・機材」といった実践知を体系的に学ぶためのLOD（Linked Open Data）モデルおよびデータセットを公開しています。

## 研究の背景
従来の音楽アーカイブでは不十分だった「演奏過程」や「実践知」の記録・共有を、ステムデータ（パート別音源）とRDFを用いることで実現しています。
